#gemini-ai-chatbot

An interactive Gemini AI Chatbot built with HTML, CSS, and JavaScript. Users can engage in conversations with the AI through a sleek interface, powered by a public API (API key removed for security). This project demonstrates secure API usage, event handling, and responsive UI design.
